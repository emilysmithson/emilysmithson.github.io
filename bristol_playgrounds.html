Bristol Playgrounds

This app is part game and part guide. It features all the very best plagyrounds in the Bristol area.

There is an interactive map and a filterable list to help you select a playground to visit.

Every time you visit a playground you win a piece of playground equipment to go in your very own virtual playground.

It uses Flare animations and the interactive map is created using Leaflet.

The character design was done by my 11 year old daughter.

Please note that the web version does not have full functionality.
